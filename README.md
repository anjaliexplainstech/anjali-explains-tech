# Anjali Explains Tech Career Hub

A mobile-first static website containing:

- Homepage
- SQL Hub
- Power BI Hub
- Resume Hub
- Interview Vault
- Friday Jobs Hub

## Preview locally

Open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Before launch

1. Replace the homepage photo placeholder with Anjali's professional portrait.
2. Replace every `ADD LINK` resource with the actual PDF, video, form or landing-page URL.
3. Connect the newsletter forms to Beehiiv, Kit, MailerLite, Brevo or another provider.
4. Add LinkedIn and business email links.
5. Populate Friday Jobs only after final verification of official application pages.
6. Add privacy policy, terms and contact page before collecting emails or selling products.

## Easy deployment

### Netlify Drop
Zip or drag the complete folder into Netlify Drop.

### GitHub Pages
Upload all files to a repository and enable GitHub Pages from the repository settings.

### Cloudflare Pages / Vercel
Import the folder or GitHub repository. No build command is required because this is a static website.

## Domain connection

Purchase the domain from a registrar using your own account and payment method. Then point the DNS records to the selected hosting platform using that platform's instructions.

Recommended first-choice domain: `anjaliexplainstech.com`, subject to live availability.
