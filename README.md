# Anjali Explains Tech Career Hub — Version 2

This version has all CSS and JavaScript embedded inside each HTML file, so the design will not disappear if the stylesheet folder is missed.

## Upload to GitHub
Upload these items to the repository root:
- index.html
- sql.html
- powerbi.html
- resume.html
- interview.html
- jobs.html
- assets folder (contains anjali-profile.jpg)

Do not upload only the ZIP. Extract it first.

## GitHub Pages
Settings → Pages → Deploy from a branch → main → /(root).
